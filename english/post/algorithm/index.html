最长无重复子串

s = ‘asdf’ l=r=0 n = len(s) d = {} res = 0 while r < n: if s[i] in dic and d[s[r]] >= l: l = d[s[i]] + 1 d[s[r]] = r res = max(res, r-l+1) r += 1 return res

最长回文子串

双指针，从头遍历 每次分单/双数讨论并向外拓展回文 def extending(s: str, l:int, r: int): while l >= 0 and r <=len(s) and s[l] == s[r]: l -= 1 r += 1

return l+1, r-1

a, b = 0, 0

for i in range(len(s)): l1, r1 = extending(s, i, i) # 单点为中心 l2, r2 = extending(s, i, i+1) # 完全对称 if r1 - l1 > b-a: a, b = l1, r1 if r2 - l2 > b-a: a, b = l2, r

return s[a:b+1]

最大容器

双指针遍历，头/尾靠拢 目标值 = 数组长度 * 两边高度取小 高度不确定，所以从长度入手 双变量定性分析，从可控的入手 high = [1,2,3,4,5,6] l = 0 r = len(list) -1 res = 0 while l < r: if high[l] < high[r]: res = max(res, high[l](r-l)) l += 1 else: res = max(res, high[r](r-l)) r -= 1
